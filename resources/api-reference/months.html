Months list

Returns the list of available months.

HTTP Request

GET /api/months
GET /api-xml/months
Query parameters
Optional parameters
return-as string
Sets the format for the result.
Set as xml to get the result as XML.
Set as json to get the result as JSON.
Default format is json.

Response

If you requested the response as JSON, the method will return a body with the following structure:

{
.. "months": [
....{
...... "id": string,
...... "year": number,
...... "month": number
....}, ...
..]
}

If you requested the response as XML, the method will return a body with the following structure:

<months>
...<month year= number month= number> YYYY-MM</month>
... ...
</months>
Properties
id string
Month identifier, formatted as YYYY-MM. For example: 2018-06.
year number
The year number. For example: 2018.
month number
The month number (01 - 12). For example: 06.
