Pokemon: Usage Data

Returns the usage details of a pokemon (abilities, moves, items...).

HTTP Request

GET /api/pokemon/data
GET /api-xml/pokemon/data
Query parameters
Required parameters
format string
Name of the format you want to get the stats from.
Example: [Gen 7] Ubers
pokemon string
Name of the pokemon you want to get the stats from.
Example: Pikachu
Optional parameters
baseline number
Ranking baseline you want to apply.
Only certain baselines are available.
If you do not specify a baseline. The default baseline is used.
Examples: 0, 1500, 1630, 1695, 1760, 1825
month string
The month when the usage stats were taken. Format: YYYY-MM
Example: 2018-10
By default, it uses the latest month.
select string
Selects the field you want to obtain.
Available fields: abilities, items, moves, spreads, teammates, counters
return-as string
Sets the format for the result.
Set as xml to get the result as XML.
Set as json to get the result as JSON.
Default format is json.

Properties
feature string
API feature you requested.
year number
Year when the usage stats were taken.
month number
Month of the year (01 - 12) when the usage stats were taken.
format string
Format name.
baseline number
Baseline used for the usage stats compilation.
id string
Pokemon identifier.
name string
Pokemon name.
rawCount number
Raw count for the pokemon.
avgWeight number
Average weight for the pokemon.
viabilityCeiling number
Viability ceiling for the pokemon.
abilities array
Most used abilities.
items array
Most used items.
moves array
Most used moves.
spreads array
Most used spreads.
teammates array
Most common teammates.
counters array
Most used counters.

Errors

Status Error Code Description
404 MONTH_NOT_AVAILABLE This error means the month you specified was not found.
Maybe you spell it wrong or the month is not available.
404 FORMAT_NOT_FOUND This error means the format you specified was not found.
Maybe you spell it wrong or the format is not available.
404 BASELINE_NOT_AVAILABLE This error means you specified a baseline that is not available.
